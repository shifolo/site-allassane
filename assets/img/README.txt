Dépose ici tes photos du shooting. Le site attend ces noms de fichiers
(tu peux renommer tes photos, ou changer les chemins dans index.html) :

- profile.jpg          -> photo de toi, utilisée dans le Hero
- project-weko.jpg      -> visuel pour la carte projet "WEKO Technologies"
- project-security.jpg  -> visuel pour la carte projet "Audit & Scan de Vulnérabilités"
- project-saas.jpg      -> visuel pour la carte projet "Application de Gestion SaaS"
- og-image.jpg          -> image affichée quand le lien du site est partagé sur
                           WhatsApp/LinkedIn/Facebook/X (format recommandé : 1200x630px)

Tant qu'une image est absente, un fond dégradé de secours s'affiche
automatiquement (pas d'image cassée à l'écran). Pour og-image.jpg en revanche,
certains réseaux sociaux n'affichent simplement rien tant que le fichier n'existe pas.
